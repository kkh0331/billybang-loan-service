# 빌려방
>`부동산`부터 `대출`까지 개인 맞춤형 올인원 서비스  

## 🔗 Link
https://github.com/PDA-BillyBang

## 📝 간단한 설명
1. 다양한 앱에 포진된 `매물`들과 `대출상품`을 모아
2. 원하는 조건으로 매물을 `검색`할 수 있고
3. 내 재정상황에 맞는 대출상품을 `추천`해주는 웹 어플리케이션

## 🙋‍♂️ 담당한 기능
- 대출 서비스와 관련된 데이터 수집 및 api 개발
